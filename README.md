# V-Headphone

[Demo](https://0445814.github.io/V-Headphone/#/)

## 專案說明

本專案是由 Vue Cli 4 所建立的 SPA 電商網站，亦為學習六角學院 Vue.js 課程後所完成的最終作品。

- 製作時間: 1 個月。
- 應用技術:
  + Vue.js
  + Vue router
  + Vuex
  + 串接 Firebase
  + 串接 API
  + 其他第三方插件等
  + Bootstrap 4
- 其他:
  + 主要以 ES6 撰寫
  + ESlint 風格規範
